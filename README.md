# EKS Hybrid Engine

A simple, immersive, and lightweight FiveM script that simulates hybrid vehicles by **switching engine audio hashes** based on speed. Below a set threshold the car uses an electric-style audio; above it, the car swaps to a gas engine audio. Smooth transitions included.

## Features
- Electric at low speed, gas at higher speed
- Built-in hysteresis to prevent rapid toggling
- Client-only, no performance footprint!
- MPH/KMH support

## Installation

1. Drag and drop the folder into your `resources/` directory  

2. Add the following to your `server.cfg`:

ensure EKS_Hybrid

## FAQ
